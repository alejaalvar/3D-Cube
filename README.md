# 3D Rotating Cube

An interactive 3D cube rendered in a 2D window using PyGame. The cube is built from scratch using matrix math — 3D points are rotated via rotation matrices and projected down to 2D using a projection matrix.

## Requirements

- Python 3.10+
- PyGame

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/your-username/cube.git
   cd cube
   ```

2. Install dependencies:
   ```
   pip install pygame
   ```

## Usage

```
python main.py
```

## Controls

| Key | Action |
|-----|--------|
| `A` / `Left` | Rotate left (Y axis) |
| `D` / `Right` | Rotate right (Y axis) |
| `W` / `Up` | Rotate up (X axis) |
| `S` / `Down` | Rotate down (X axis) |
| `Q` | Rotate counterclockwise (Z axis) |
| `E` | Rotate clockwise (Z axis) |
| `R` | Reset rotation |
| `Ctrl+C` | Quit |
